# ZigSelector

Simple user environment path selector for [Ziglang](https://ziglang.org/)

Simply add binaries in same directory and set it **config.json**

![prooit](https://i.ibb.co/vX6SzpG/zigselector.jpg)

## args
 - --help
 - --add-to-path
   
## publish
 - net 8.0
 - win-x64
 - produce single file
